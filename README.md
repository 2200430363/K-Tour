# K-Tour
K-Tour is an Android-based tourism guide application developed to provide complete and organized tourist information about Kolhapur city.
The application helps tourists explore Kolhapur by offering categorized information about historical places, religious sites, natural attractions, cultural landmarks, food destinations, hotels, and nearby places.
It is designed with a simple and user-friendly interface to make travel planning easy and convenient.
#Features
Tourist place information
Categorized destination browsing
Historical and religious site details
Hotel and restaurant information
Nearby places guidance
Simple search functionality
Easy-to-use interface
#Technologies Used
Java
XML
Android Studio
SQLite Database
